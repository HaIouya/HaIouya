Waline 评论系统部署指南

本文将指导您如何从零开始部署 Waline 评论系统，并将其集成到您的 Hexo 博客中。

什么是 Waline？

Waline 是一个基于 Valine 衍生的简洁、安全的评论系统，支持 Markdown、表情、通知等功能，并且完全开源免费。与 Valine 不同，Waline 需要服务端部署，但这也带来了更好的数据安全性和功能扩展性。

部署步骤

1. 准备工作

您需要准备以下内容：

  • 一个 GitHub 账号
  • 一个 Vercel 账号（可以直接用 GitHub 账号登录）
  • 一个数据库（LeanCloud、MongoDB、MySQL、PostgreSQL、SQLite 或 GitHub）

2. 部署 Waline 到 Vercel

  1. 访问 Waline 官方仓库
  2. 点击 “Deploy with Vercel” 按钮
  3. 使用 GitHub 账号登录 Vercel
  4. 为您的项目设置一个名称，然后点击 “Deploy”
  5. 部署完成后，Vercel 会为您的 Waline 服务生成一个域名，如 https://your-project-name.vercel.app

3. 配置环境变量

在 Vercel 的项目设置中，添加以下环境变量：

使用 LeanCloud 作为数据库（推荐新手使用）

  1. 注册并登录 LeanCloud 国际版
  2. 创建一个应用，进入应用的设置页面
  3. 在应用凭证中获取 APP ID、APP KEY 和 Master Key
  4. 在 Vercel 中添加以下环境变量：
    • LEAN_ID: 您的 LeanCloud APP ID
    • LEAN_KEY: 您的 LeanCloud APP KEY
    • LEAN_MASTER_KEY: 您的 LeanCloud Master Key

使用 MongoDB 作为数据库（适合有经验的用户）

  1. 注册并登录 MongoDB Atlas
  2. 创建一个免费的 Shared Cluster
  3. 在 “Database Access” 中创建一个用户并设置密码
  4. 在 “Network Access” 中添加 IP 地址 0.0.0.0/0（允许所有 IP 访问）
  5. 获取连接字符串，格式为 mongodb+srv://<username>:<password>@<cluster-url>/<database-name>
  6. 在 Vercel 中添加环境变量：
    • MONGODB_URI: 您的 MongoDB 连接字符串

4. 添加评论管理员

  1. 在 Vercel 环境变量中添加：
    • ADMIN_EMAIL: 管理员邮箱，可以添加多个，用逗号分隔

使用与管理

评论管理

访问 https://your-project-name.vercel.app/ui/register 注册管理员账号（使用配置的管理员邮箱）。
登录后，可以在 https://your-project-name.vercel.app/ui 管理所有评论。

常见问题

Q: 评论无法显示怎么办？

A: 检查浏览器控制台是否有错误信息。确认 serverURL 设置正确，并且 Waline 服务正常运行。

Q: 如何自定义评论样式？

A: 可以通过在博客中添加自定义 CSS 来修改 Waline 的样式。

Q: 评论数据安全吗？

A: Waline 的数据存储在您自己控制的数据库中，比第三方服务更安全。建议定期备份数据库。
